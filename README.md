# 📓 My Personal Notes App

Aplikasi catatan pribadi sederhana namun fungsional yang dibangun menggunakan **React.js** dan **Vite**. Proyek ini dibuat untuk mengelola catatan harian dengan fitur pencarian yang responsif.

---

## 🚀 Fitur Utama

* **Tambah Catatan:** Simpan ide atau tugas penting kamu dengan mudah.
* **Hapus Catatan:** Bersihkan catatan yang sudah tidak diperlukan.
* **Arsipkan Catatan:** Pisahkan catatan aktif dan catatan yang sudah selesai.
* **Fitur Pencarian:** Cari catatan tertentu secara *real-time* berdasarkan judul.
* **Responsive Design:** Tampilan yang nyaman diakses dari perangkat apapun.

## 🛠️ Tech Stack

* **Frontend Library:** [React.js](https://reactjs.org/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Styling:** CSS3 (Custom Styles)
* **Language:** JavaScript (ES6+)
